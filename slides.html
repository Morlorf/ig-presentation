  • Panoramic train circling the park

CODE STRUCTURE

  • Domain folders: environment/ · rides/ · people/ · controls/ · lighting/ · ui/
  • App.js — scene, renderer, post-processing (UnrealBloomPass), animation loop
  • EventBus — pub/sub communication between components
  • No build step — vanilla ES modules + importmap

THREE HARD PROBLEMS

Water Shader

  • 4 Gerstner waves summed in vertex shader
  • Analytical normals from partial derivatives
  • Noise-based caustics on surface
  • Lighting reacts to every ripple
GIF: 02-waves.mp4

Coaster Track

  • CatmullRom spline from rail-tube mesh
  • Rotation-minimizing frame (unlike Frenet, no flip at inflection)
  • Energy-conservation speed model
  • Sideways banking on curves

NPC Gait

  • A* pathfinding on navigation grid
  • String-pulling path smoothing
  • Zero foot-skate (foot slides backward at walk speed)
  • 2-bone IK solving the knee

RIDES & LIGHTING

Ride System

  • All rides extend RideBase.js
  • State: start() / stop() / toggle()
  • Speed ramping via tickSpeed() + ControlPanel
  • Bloom layers via applyBloomLayers()
  • EventBus listeners for cross-component events

Lighting

  • Sun & moon orbit on celestial sphere
  • Sky: 4 HDR textures crossfaded via GLSL mixing
  • Lampposts: Auto / On / Off — clickable, automatic at sunset

FERRIS WHEEL & CAROUSEL

Ferris Wheel

GIF: 07-ferris-fpv.mp4
  • Each gondola in a separate group
  • Group counter-rotates to cancel wheel rotation
  • Solved with quaternion multiplication
  • Passengers stay upright

Carousel

GIF: 08-carousel-fpv.mp4
  • Each horse has unique phase offset
  • Phase-shifted sine waves
  • Creates a traveling wave effect around the carousel
  • Up/down animation per horse

ROLLER COASTER & TAGADA

Roller Coaster

GIF: 09-coaster-fpv.mp4
  • Track extracted from GLB rail-tube mesh rings
  • Each ring → control point on CatmullRom spline
  • Rotation-minimizing frame → no flipping
  • Train speed: accelerates downhill, slows uphill
  • Sideways banking on curves

Tagada

GIF: 10-tagada-fpv.mp4
  • 6 simultaneous movements:
  • Platform spins · arm yaws
  • Arm pitches · arm rolls
  • Bounces on 2 axes · arm telescopes outward
  • Signature chaotic feel

SHOOTING GALLERY

  • T — enter aiming mode (first-person + crosshair)
  • 30-second timer
  • Raycasting from crosshair — instant hit detection, no physics
  • Close targets: slower movement, fewer points
  • Far targets: faster movement, more points
  • ESC — exit aiming mode
